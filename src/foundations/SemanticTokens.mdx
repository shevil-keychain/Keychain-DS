# Semantic Tokens

Role-based tokens that components consume directly — never raw primitives like `bg-gray-50`
or `rounded-8`. Each token has a description and a "when to use" line; AI agents and humans
should both read those before reaching for primitives.

Source of truth for descriptions: [`src/tokens/semantic.ts`](../tokens/semantic.ts).
